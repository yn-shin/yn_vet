# Overview

- 질소 노폐물의 excretion

#### Urinary tract(비뇨기계)의 구성

- **Kidney (콩팥, 신장)** : 소변 생성
- **Ureter (요관)** : 신장에서 생성된 소변을 방광으로 운반
- **Bladder (방광)** : 소변 저장
- **Urethra (요도)** : 소변을 외부로 배출하는 마지막 경로
