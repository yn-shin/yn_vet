- 약물요법은 항암치료의 극히 일부일 뿐이며, 보통 5개 이상의 약을 병용한다.
- 항종양약물의 일반적인 부작용으로는 구토, 골수억제, 탈모가 있다. 빠르게 증식하는 세포를 target으로 하기 때문이다.

# 항대사물 (Antimetabolites)

- RNA나 DNA 합성에 있어서 정상적인 화합물에 대해 경쟁하거나 합성을 억제함으로써 정상적인 purine/pyrimidine nucleotide 전구물질의 이용을 억제한다.
- 세포주기 특이성을 갖는다.

| 약물                      | 내용                                                                                                                                               | 부작용                                      |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------------------------- |
| methotrexate            | - 엽산 대사를 차단시킴<br/>- 림프종, 유방염, 방광암 등 다양한 암에 효과, 낮은 용량의 MTX는 면역 억제 기능이 있어 심한 건선이나 류마티스성 관절염 치료에도 사용<br/>- BBB를 스스로 통과하지 못함.                        | N/V/D(구역/구토/설사), alopecia(탈모)            |
| 6-Mercaptopurine (6-MP) | - purine의 항대사물로, Crohn씨병의 치료에 유용                                                                                                                 | N/D/V, anorexia, myelosuppression        |
| fludarabine             | - purine nucleotide 유사체<br/>- 만성 림프구성 백혈병, 모발세포성 백혈병 치료<br/>-위장관내 세균에 의해 독성 대사물(fluoroadenine)으로 분해되기 때문에 경구투여보다는 정맥주사로 투여                       | N/V/D, immunosuppression, fevers, edemas |
| 5-Fluorouracil (5-FU)   | - pyrimidine 유사체, DNA 합성의 전구물질 thymidine을 세포에서 박탈함<br/>- 서서히 성장하는 고형종양 치료에 주로 사용함(결장직장암, 유방암, 난소암, 췌장암, 위암)<br/>- 심한 위장관 독성 때문에 정맥주사 (피부암은 국소투여) | D, alopecia, muscositits                 |
| capecitabine            | - 일련의 효소단계를 거쳐 5-FU 로 가수분해되는 prodrug<br/>- 결장직장암 및 전이성 유방암 치료                                                                                    | D, muscositis, myelosuppression          |
| cytarabine              | - pyrimidine 길항제로 작용하여 DNA의 polymerization을 억제<br/>- 급성 골수성 백혈병에 사용<br/>- 위장관 점막 및 간에서 비세포독성으로 탈아민되기 때문에 경구 투여는 효과가 없음                           | N/V/D, myelosuppression                  |

# 항종양 항생제 (Antibiotics)

- 주로 DNA 기능의 붕괴를 초래함
- topoisomerase Ⅰ, Ⅱ 억제와 free radical 생산 능력이 세포독성에 중요한 역할

| 약물                                           | 내용                                                                                                                                                                             | 부작용                                                                                                     |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| 안트라사이클린류<br/>( doxorubicin, daunorubicin, idarubicin, epirubicin, mitoxantrone ) | - doxorubicin 은 가장 중요하고 널리 사용되는 항암제로, 육종, 유방암, 급성 림프구성 백혈병 치료 시 병용함<br/>- daunorubicin, idarubicin 은 급성 백혈병 치료<br/>- 위장관 내에서 불활성화되므로 IV 투여, BBB 통과 불가<br/>- 비가역적 용량의존성 심상독성 | N/D/V, myelosuppression, alopecia, cardiac toxicity<br/>- doxorubicin, daunorubicin 에서 부작용이 심한 편<br/> |
| bleomycin                                   | - 구리 착화합물-glycopeptides 혼합물, free radical이 DNA 절단<br/>- IV/SC/IM 투여 가능                                                                                                         | pulmonary fibrosis, alopecia                                                                            |

# 알킬화제 (Alkylating agents)

- 친핵 그룹과 공유 결합을 하면서 세포독성 효과를 나타냄
- 신속히 분열하는 세포에 가장 큰 독성을 나타냄
- DNA freezing으로 인해 정상세포 또한 손상을 입음

| 약물                            | 내용                                                                                                 |
| ----------------------------- | -------------------------------------------------------------------------------------------------- |
| cyclophosphamide & ifosfamide | - 대사 물질이 DNA와 반응하여 세포 독성을 나타냄<br/>- 부작용은 출혈성 방광염, 방광 섬유증                                           |
| nitrosourea                   | - CNS 침투 능력이 있어 뇌종양 치료에 사용<br/>- DNA 알킬화로 복제를 억제함으로써 RNA와 단백합성을 억제함"                               |
| dacarbazine,<br/>temozolomide | dacarbazine 은 melanoma(흑색종) 치료<br/>temozolomide 는 BBB 통과 능력이 있어 뇌종양(glioblastomas, astrocytoma) 치료 |
| mechlorethamine               | 림프암                                                                                                |
| melphalan                     | 다발성 골수암                                                                                            |
| chlorambucil                  | 만성 림프구성 백혈병                                                                                        |
| busulfan                      | 만성 골수성 백혈병                                                                                         |

# 미세관 억제제 (Microtubule inhibitors)

- 유사분열 방추체에 영향을 미쳐 세포독성을 일으킨다.
- 전부 정맥내로 투여한다.

| 약물                                                         | 내용                                                                                                                                                       |
| ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| vincristine(VX),<br/>vinblastine(VBL)<br/>vinorelbine(VRB) | - tubuline이 미세관을 형성하기 위해 중합되는 작용을 차단<br/>- 혈관 외로 유출되면 정맥염, 구토, 설사, 탈모증 유발"                                                                               |
| paclitaxel,<br/>docetaxel                                  | - 세포주기의 G2/M기에 활성, 미세관의 축적을 일으켜 기능이 없는 미세관을 생성<br/>- paclitaxel 은 난소암, 전이성 유방암, 비소세포 폐암 치료<br/>- docetaxel 은 전립샘, 유방, 비소세포 폐암<br/>- 부작용은 호중구감소증 및 백혈구감소증 |
