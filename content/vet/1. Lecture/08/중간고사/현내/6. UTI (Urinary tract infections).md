# Urinary tract infections (비뇨기계 감염증)
# 1. Introduction

>[!check] Urinary tract (비뇨기계)
>- **신장(kidneys)** : 소변 생성
>- **요관(ureters)** : 신장에서 생성된 소변을 방광으로 운반, 양쪽으로 2개
>- **방광(bladder)** : 소변을 저장하는 주머리
>- **요도(urethra)** : 방광의 소변을 외부로 배출하는 마지막 경로
### Urinary tract의 defense mechanisms 

| Defense mechanism       |                                                                                                                                            |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| Normal urination        | - Adequate flow<br>- Complete, frequent voiding (배뇨)<br>    : 세균 정체 없이 제거                                                                  |
| Anatomical structures   | - **Lengt**h of urethra<br>- **Urethral high pressure** zone<br>   : 요도근위부 주름 팽창으로 압력 상승<br>- Urethral peristalsis                         |
| Antimicrobial urine     | - **High urea content**<br>- **Low urine pH** : 균 성장 억제<br>- **Osmolarity**<br>- Organic acids<br>- Tamm-Horsfall protein                  |
| Mucosal defence barrier | - **Balanced distal urethra microflora**<br>- **Surface IgA antibodies**<br>- Glycosaminoglycan layer<br>- Exfoliation of urothelial cells |
- 이러한 방어 체계가 깨지면 감염에 취약해짐.
## Urinary tract infections

- 외부 환경에서 bacteria가 urethra(요도)로 들어가, 신체의 면역 방어 체계를 깨뜨리고 uniray tract(비뇨기계)에 감염을 유발한 상태.

# 5. 치료
- 방광 종양, 호르몬 질환 등 배제하고 시작. (혹은 같이 관리)
- 면역억제 약물을 사용하는 환자에서, 비뇨기계와 피부 감염에 취약

- **NSAIDs** : 신독성 존재, 유의해서 처방.

### Subclinical bacteriuria가 있는 경우
- 불필요한 항생제 처방보다는, 모니터링이 권장됨.

### Uncomplicate UTI
- 적합한 항생제를 처방하면 이틀 안에 개선이 보임.
- 이 개선이 보이지 않으면 complicate UTI거나, 항생제를 잘못 썼거나.
- cytology를 했을 때 3-5일 후부터 개선됨.
- 방광염 환자에서 하루 1번 약을 먹이는 경우, 저녁에 먹이는 것을 권장함. (수면 시간 동안 충분히 약물 작용할 수 있도록)

- 경험적 항생제를 우선 처방해야 하는 상황도 있음.
- Cytology → 간균, 단간균이 다수 관찰된다


|         |                                                           |                                                |
| ------- | --------------------------------------------------------- | ---------------------------------------------- |
| 간균, 단간균 | E. coli,<br>Enterobacter,<br>Klebsiella,<br>Proteus, etc. | **Cephalosporins**<br>우선적으로 고려                 |
| 구균      | Staphylococcus,<br>Streptococcus,<br>Enterococcus, etc.   | **Amoxicillin**, **Cephalospotin**<br>우선적으로 고려 |
- **Amoxicillin**
- **Trimethoprim-sulfamethoxazole** 
- Amoxicillin clavulanate
- Enrofloxacin : 1루 1번 복용 퀴놀론 제제, 침투력이 뛰어남.
	- 퀴놀론 → 내성이 잘 생김. uncomplicated 따위의 병에 너무 막 쓰면 다른 필요한 병에 쓸 수 없을지도.

#### Recurrent & Complicated Bacterial cystitis
- 2주 정도 지켜보며 써라.
- 항생제를 적절히 선택하는 것이 중요.
- 재발성 → 저항성 고려하여, monitoring 시기나 치료 반응 기간을 길게 볼 필요가 있음.
- 항생제가 진짜 안 듣는 경우, 보조 요법을 사용할 수 있음.
	- '보조 요법'인 만큼, 단독으로 치료하기엔 적절하지 않음.
- 개의 피부는 염기성/중성인 경우 많음. 피부에 세균 감염이 잘 생김.
- 비뇨기계도 사람보다 개에서 acidic하게 소변을 바꾸는 게 어려워서, 보조제 효과가 사람에 비해 약함.
- 소변이 너무 산성이 되면 항생제 response가 저하

>[!note] Cranberry proanthocyanidins (PACs)
>- 개나 고양이가 크랜베리 맛을 좋아하지 않아, 파우더 제제로 사용
>- 대장균의 부착을 억제하는 효과가 있다.

### Treatment failure
>[!danger] 치료 실패의 이유
>- 부적절한 치료
>- Impaired action of drugs (보호자가 약을 잘 먹이지 않아서)
>	- 정 어려우면 2주 정도 가는 주사 제제로 교체 등
>- 내성 견주, 
>- **AMR (antimicrobial resistance)** : 항생제 저항성 심각
>	- MDR (Multidrug-resistant) bacteria : 3개 이상의 카테고리에 감수성 있는 게 한두개는 있음. (퀴놀론 제제, 등)
>	- XDR (Extensively drug-resistant) bacteria : 그보다 내성 패턴이 더 강함.
>	- PDR (Pandrug-resistant) bacteria : 사람에서만큼 많지는 않음. 
>- β-lactam 계열 효과 없다고 알려짐.
>- Carbapenem : 
>- 퀴놀론 제제, amino, carbapenem 등

- 수액 조치가 치료에 좋음
- 신우신염의 
- 퀴놀론 제제, 없으면 amoxicillin
- cefovecin

- 절제가 더 근본적인 치료

- 신우신염, 방광염에 비해 좀 더 침투력이 좋은 약물이 필요함.
